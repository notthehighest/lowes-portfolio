06 — Flagship Project

MAISCAN

A closer look at the project I'm most proud of.

Machine Learning-Based Corn Classification & Sorting System

MAISCAN is an innovative technology project developed to improve efficiency and accuracy in corn classification and sorting. The system uses machine learning and computer vision to automate the identification and categorization process, paired with a user-friendly web-based interface for monitoring and management.

My contributions covered project leadership, research development, system analysis and design, frontend development, documentation, and presentation/demonstration of the system. Key impact: reduced manual classification effort, improved sorting consistency, and demonstrated a practical application of AI in agriculture.
